# CRD-Operation-Using-Node
Build a REST-API based server to easily create, delete, and read data from a JSON file.

## Local Setup

### Requirements
- Nodejs;
- Express;
- NPM;
- Chai;
- Mochi;
- Supertest;


## Installing and running

**Git:**
```bash
git clone

npm install
npm start
```
